# monarch-tools

Clean-slate Phase 1 scaffolding for the Monarch Tools project.

## Run

From the repo root:

python3 -m monarch_tools hello
python3 -m monarch_tools name Keith
python3 -m monarch_tools help
python3 -m monarch_tools extract chase sample.statement.pdf

`extract` is a stub in Phase 1 and only prints what it would do.
